🚀 QML HMI 開發進階之路
第一階段：QML 核心語法 (基礎功)
這一階段目標是熟悉 QML 的宣告式語法與布局邏輯。

L1：元素與屬性 (Elements & Properties)

Rectangle, Text, Image 的基本使用。

自定義屬性 (property) 與屬性綁定 (Binding) 的威力。

L2：佈局藝術 (Anchors & Layouts)

Anchor 定位法（Top, Bottom, Center…）。

Row, Column 與 Grid 的使用場景。

L3：互動與輸入 (Interaction)

MouseArea 與觸控事件處理。

訊號與槽機制 (Signals & Handlers) 的基礎。

第二階段：動態效果與視覺美化 (HMI 的靈魂)
利用 Jetson Nano 的 GPU 來實現流暢的 UI。

L4：動畫與過場 (Animations & States)

PropertyAnimation 與 NumberAnimation。

使用 State (狀態) 與 Transition (過渡) 製作多頁切換。

L5：畫布與圖形效果 (Canvas & Graphical Effects)

使用 QtGraphicalEffects 製作陰影、模糊與發光。

Canvas 繪圖：手繪一個簡單的儀表板刻度。

第三階段：邏輯與數據處理 (後端連接)
學習如何將 UI 與邏輯分離。

L6：清單與模型 (ListView & Models)

使用 ListModel 與 Delegate 顯示多組數據（例如：多顆電池單體電壓）。

L7：C++/Python 混合編程 (Context Properties)

將 C++ 或 Python 的變數「注入」到 QML 中。

核心挑戰：從後端每秒更新一次隨機數值，並同步顯示在 QML 儀表上。

第四階段：Jetson Nano 實戰專案 (硬體整合)
這部分會觸及你最熟悉的領域，但改用 QML 呈現。

L8：CAN Bus 數據視覺化

讀取 SocketCAN 數據並即時更新到 QML UI。

L9：嵌入式系統優化 (Performance)

學習使用 nveglglessink 嵌入硬體加速影片。

EGLFS 模式調試：在沒有 X11 桌面下的全螢幕運行。
