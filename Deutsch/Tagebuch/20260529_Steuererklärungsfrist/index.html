Steuererklärungsfrist (29. Mai)

這句怎麼說(Wie sagt man das auf Deutsch)

一、「我發現」怎麼說

1. Ich habe festgestellt, dass …: 正式、客觀地發現（工作、分析）

2. Ich habe gemerkt, dass …: 口語、突然察覺到

  • Ich habe gemerkt, dass …
  • Ich habe gemerkt, dass ich mein Handy vergessen habe.

3. Jetzt habe ich festgestellt, dass …: 「原來…」的感覺

4. Jetzt weiß ich, dass …: 「原來…」的感覺

二、「超爽的」怎麼說

1. Das ist echt geil!: 現代口語用來表示「超爽的」。geil原意是「性興奮的」

  • Wie geil ist das denn?!
  • 這也太爽了吧？！

2~4比較安全、人人都能用

2. Das ist super!

3. Das ist großartig!

4. Das ist fantastisch!

三、「…之後，我…」怎麼說

Nachdem …, habe ich …

日記內容 Tagebuch (29. Mai)

Gestern war 29. Mai,
der drittletzte Tag vor der Steuererklärungsfrist.

Nachdem ich meine Steuererklärung gemacht habe,
habe ich festgestellt,
dass ich keine Steuern zahlen muss.

Jetzt weiß ich,
dass ich wohl arm bin.

Das ist echt geil!

翻譯

昨天是5月29日，報稅截止的前三天
當我報稅之後，我發現，我不用繳稅
現在我知道我很窮
超爽的

📘 die Frist: 期限、截止日期

📘 Einkommensteuer: 所得稅
