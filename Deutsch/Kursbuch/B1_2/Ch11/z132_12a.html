einen Stadtbummel machen
→ 在城市裡散步／逛街

einen Einkaufsbummel machen
→ 逛街購物

an einer Rundfahrt teilnehmen
→ 參加觀光導覽／城市巡遊

auf einen Turm steigen
→ 登上一座塔

ein Fahrrad mieten
→ 租腳踏車

in die Oper gehen
→ 去歌劇院／看歌劇

ein Museum besuchen
→ 參觀博物館

eine Kirche besichtigen
→ 參觀教堂

in den Bergen wandern
→ 在山區健行

ein Denkmal anschauen
→ 觀看紀念碑／古蹟

in den See springen
→ 跳進湖裡

mit dem Boot fahren
→ 搭船／乘船

in der Fußgängerzone shoppen
→ 在徒步區購物
